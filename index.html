SOBRE MIM

Sou de Toledo, tenho 23 anos e faço faculdade de engenharia de computação pela UTFPR, estou no 6º periodo. Criei este projeto de portifólio tanto para treinar minhas habilidades com HTML e CSS quanto para apresentar os meus projetos que já fiz. Tenho conhecimento em C/C++, Python, Dart/Flutter, VBA, SQL, HTML e CSS.

Campo Minado

Este foi meu primeiro grande projeto na faculdade, a idéia era construir um jogo de campo minado em C que fosse possível jogar, salvar e carregar para jogar novamente outra vez que fosse executado. Não é usado interfece gráfica, então ele roda pelo terminal. É possivel ver mais sobre o funcionamento dele através do executável e o código fonte.

Sistema de Portas Lógicas

Projeto de orientação a objetos feito em C++ através da SDK QtCreator. O programa foi criado 100% sem ajuda de Forms. É um software que simula um sistema de portas lógicas que pode ser montado de acordo com a sua necessidade, foram implementados os operadores lógicos or, and e not. O programa ainda permite salvar e carregar os sistemas criados. Você pode fazer o download do programa para utilizar e ter acesso ao repositório disponível aqui.

Análise Musical - Método de Agrupamento K-Means

Um programa de ciência de dados, escrito em Python, feito para agrupar músicas por gênero. Os dados foram recolhidos através da API do Spotify, então foi feito uma análise e tratamento de dados para determinar duas caracteriscas que poderiam determinar o gênero da música. Para determinar os clusters foi utilizado o algoritmo de mineração de dados K-Means. Eu escrevi um artigo explicando melhor sobre o projeto, também é possivel visualizar o código fonte.

Zodiaco Karma

Este é um app que fiz em Flutter, o app mostra o carma de acordo com o signo da pessoa, é algo simples que usei para aprender a usar dropdownbox, mudar nome e icone do apps. Acesso ao repositório aqui.

Dado da Sorte

App feito em flutter, fiz para testar o uso de GestureDetector, é um simples jogo de dados com 50/50 chances de vitória. A idéia é futuramente inserir vários jogos de azar. O repositório dele pode ser acessado aqui.
